Max of Two Numbers Assignment

Description of Assignment
  • Problem: write a function that takes two numbers and returns the maximum of the two numbers
  • Three test cases exist:
    • Number A > Number B
    • Number A < Number B
    • Number A = Number B
  • Use strings to return calculated answer
    • Examples: larger, smaller, equal

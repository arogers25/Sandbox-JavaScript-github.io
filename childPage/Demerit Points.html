Demerit Points Assignment

Description of Assignment
  • Problem:input a speed and calculated the demerit points
  • Bonus (Advanced): also input the speed limit
  • Note: speed limit is 70 km per hour
  • Note: for every 5km above speed limit, 1 demerit point
  • Note: at any speed limit, 12 demerit points means license suspended
  • Test cases:
    • 68 km/h: 0 points
    • 70 km/h: 0 points
    • 71 km/h: 0 points
    • 74 km/h: 0 points
    • 75 km/h: 1 point
    • 85 km/h: 3 points
    • 102 km/h: 6 points (round down)
    • 115 km/h: 9 point
    • 130 km/h: license is suspended
  • Coding Hints
    • math.floor(): function to remove any decimal places, never rounding
